# 👋 Вітаємо на офіційній вікі Градієнту!

{% hint style="success" %}
Початок серверу: 20.12.2022 | Офіційний запуск: 25.12.2025
{% endhint %}

{% hint style="info" %}
Айпі гри: gg.gradient-mc.com | Версія 1.21.1+
{% endhint %}

Проєкт gradient-mc - це український простір мінізабавок, де можна відволіктися від рутини, позмагатися з друзями та просто круто провести час. Від напружених битв до чилових режимів - ми намагаємося створити місце, де кожен знайде гру до душі.

Це проєкт, який розробляється з величезною любов'ю до гри. Головне правило тут: «Створено мною - покращено тобою». Сервер постійно розвивається, і левова частка крутих ідей приходить саме від нашої спільноти. Тому ми завжди відкриті до ваших пропозицій та фідбеку!

📚 Для чого ця Вікі? Щоб ви не витрачали час на пошук базової інформації, ми зібрали тут усе найнеобхідніше. Гайди по режимах, правила, фічі сервера та відповіді на часті запитання — все це тепер під рукою, щоб ви могли зосередитися на головному: грі та перемогах!
